虚拟现实

三维虚拟现实是数字城市规划、建设、管理的核心技术。数字城市综合运用GIS、遥感、虚拟现实仿真等高科技技术，融合网络、多媒体等多方数据，将城市建设可视化，为城市规划提供辅助支持。三维数字建筑模型有别于物理模型的单一俯视角，可任意角度观察，更全面、真实的展现建筑体量、园区形态及景观布局等要素，并且可提供日景、夜景、素模、室内等多样的模型效果。

城市级地上建筑虚拟现实数据技术服务----Virtual reality data technology services on city level

基于航拍图、总平面图等数据资料，综合运用GIS、遥感、网络、多媒体及虚拟现实仿真等高科技技术，对城市区域进行一比一高精度仿真现实虚拟建模。依托三维虚拟现实平台，为数字城市规划管理提供可视化的方法。 城市级虚拟现实通过显示不同建筑方案同周边建筑群体的相互关系，真实再现规划建筑和现状建筑的空间关系，能适时修改方案，使得规划评审专家和决策者可以从多个观察角度直观的对比多个规划设计方案，帮助规划决策者更加清楚直观地确认合理方案，从而提高规划管理水平，减少决策失误和盲目性，提高规划评审质量和提升评审决策的科学性，使规划管理更合理、更科学、更透明。

单体建筑项目虚拟现实数据技术服务----Monomer building project virtual reality data technology service

基于项目前期设计方案，对设计方案数据进行三维可视化呈现，为客户提供虚拟现实数据技术服务。 虚拟现实数据技术使建设单位及设计师一目了然地了解项目设计原理及竣工效果，颠覆了传统的二维平面图及效果图可视化手段，大大提高了方案设计的科学性、时效性。同时在房地产开发中的其他重要环节包括申报、审批、设计、宣传等方面都有着非常迫切的需求。

室内建模----Indoor modeling

我们为室内设计提供的服务包括：

1、直接根据客户的要求和提供的资料，为客户制作室内虚拟样板间。

2、为室内设计相关企业提供三维互动的室内设计方案。

1.地下空间虚拟现实数据技术服务----Underground space virtual reality data technology services

虚拟现实技术，运用GIS定位系统、激光扫描点云数据采集等高科技技术手段，对地下建筑进行虚拟现实高精度建模。虚拟现实数据技术服务可实现随人视角漫走其中的效果，清晰地了解地下空间结构布局，提高对地下空间开发与利用的科学性、准确性，地下空间虚拟现实数据技术服务在地下空间选址、地下工程建设、地下空间管理等领域起到至关重要的作用。

2.地下管线虚拟现实数据技术服务----Virtual reality data technology services for underground pipelines

通过运用三维市政可视化驱动软件、虚拟现实仿真技术呈现地下管线的三维场景，清晰直观地展现地下管线的阀门、井盖、消防栓、泵站、电力箱等附属设施的准确坐标位置，为城市动态地下管线信息管理提供便利，有利于实现地下管线数据的制度化、科学化、规范化和标准化。

用户体验类（DMS）数据增值--User experience class (DMS) data value added

廊坊数字空间科技有限公司是中国专业从事虚拟现实、三维互动仿真平台技术的公司。公司成立以来，将虚拟现实技术广泛应用于社会房产规划设计行业，提供数据定制服务业务，对区域范围内的城市三维数据模型进行细化、加工和再包装，用于对外宣传、展示、招商。

我公司为开发建设单位提供楼盘样板间室内三维建模业务，搭建多元化查询模式，让用户体验身临其境的美妙感觉，客户体验也将更加深入和个性化。利用空间信息构筑虚拟平台，将包括城市自然资源、社会资源、基础设施、人文、经济等有关的城市信息，以数字形式获取并加载上去，从而为政府和社会各方面提供广泛的服务。

效果图制作----Effect drawing

景观环境效果图、建筑效果图、室内效果图是根据二维AutoCAD数据及设计师设计思路，通过3D MAX、Sketch-Up、Photoshop等软件制作三维模型，采用VR渲染技术使方案达到可视化立体效果。

视频制作----Video production

可实现建筑景观视频及宣传类多媒体视频制作。

建筑景观视频制作通过路径设置、视频剪辑等技术，模拟人视角来表现建筑项目的结构设计、景观设计、室内设计、项目配色设计以及周边配套生活设施等。更加真实地体验园区的形态、布局和景观效果。

宣传类多媒体视频涉及企业形象片、产片宣传片及汇报片等。

企业VI形象设计----Enterprise vi image design

可根据客户需求提供定制化、专业的logo设计、VI设计、品牌设计、卡片设计及画册/宣传册设计相关产品制作。 VI即(Visual Identity)，通译为视觉识别，是CIS系统中最具传播力和感染力的层面。人们所感知的外部信息，有83%是通过视觉通道到达人们心智的。
