管理系统

软件开发是根据用户要求建造出软件系统或者系统中的软件部分的过程。软件开发是一项包括需求捕捉、需求分析、设计、实现和测试的系统工程。是针对政府规划管理部门开发的规划管理综合服务平台，通过可视化、图文一体的方式，实现了对“一书三证”行政许可类审批流程以及规划设计条件、修建性详细规划、建设工程设计方案、竣工验收等主要业务流程的在线审批。

我们在计算机软件、数据库技术、GIS地理信息系统、网络通讯技术等行业拥有较为丰富的经验，对规划信息化领域有较深的研究与实践，根据用户需求，提供定制化专业技术服务和产品支撑。数字规划管理系统在提高规划管理工作效率的同时，可及时对审批过程中产生的文件进行数字化归档，为规划管理部门建立电子档案提供服务。我们致力于市区空间信息的基盘建设，推进空间信息平民化，立志于以优良的产品、多样化的信息、专业的服务，演绎出空间信息新文化，争取行业内的领先。

管理系统(MIS)----Management system (MIS)

管理信息系统 ，是一个由人、计算机及其他外围设备等组成的能进行信息的收集、传递、存贮、加工、维护和使用的系统。

我公司开发的管理信息系统（MIS），旨在对日常工作事务的操作进行管理。系统主要用于管理需要的记录，并对记录数据进行相关处理，将处理的信息及时反映给管理者的一套网络管理系统。企业通过MIS系统的使用可以为企业带来巨大的经济效益和社会效益，企业的办公及管理都将朝着高效、快速、无纸化的方向发展。

地理信息系统(GIS)----Geographic information system (GIS)

地理信息系统是一种特定的十分重要的空间信息系统，它是在计算机硬、软件系统支持下，对整个或部分地球表层（包括大气层）空间中的有关地理分布数据进行采集、储存、管理、运算、分析、显示和描述的技术系统。

CAD辅助审批系统--CAD auxiliary examination and approval system

CAD辅助审批系统主要用于审批部门工作人员对现有图形资料的查询与项目红线的绘制。结合地理信息系统中的空间数据库，工作人员不但可以通过CAD辅助审批系统查看现有的各类CAD图形资料。工作人员可在AutoCAD的环境下完成项目红线的绘制工作，绘制完成后可直接保存到规划审批资料数据库中，供其他系统调阅使用，实现了GIS平台与CAD平台的互通。

数字规划管理系统----Digital planning management system

数字规划管理系统根据建设项目审批的工作流程，帮助规划管理部门业务人员实现报建项目从窗口收件到业务科室办理、领导审核审定、归档、制证、批后管理等动态流转、逐级审批全过程数字化。

数字规划管理系统在提高规划管理工作的效率的同时，审批过程中产生的文件，可及时进行数字化归档，为规划管理部门建立电子档案室提供服务。

政府企业OA、网站----Government enterprise OA, website

OA(办公自动化)系统是面向组织的日常运作和管理，员工及管理者使用频率最高的应用系统。OA在应用内容的深度与广度、IT技术运用等方面都有了新的变化和发展，并成为组织不可缺的核心应用系统。OA系统的使用也推行了一种无纸化办公的模式。

我公司开发的政府企业OA系统采用.NET与JAVA的技术路线，基于工作流的概念，使政府或企业内部人员方便快捷地共享信息，高效地协同工作；改变过去复杂、低效的手工办公方式，实现迅速、全方位的信息采集、信息处理，为企业的管理和政府的决策提供科学依据。

我公司可针对客户需求制作开发各种原创或样板的专业性网站。可根据用户需求采用ASP、PHP、C#等语言进行网站开发。通过“听取需求”、“规划设计”、“制作建设”、“测试发布”、“维护管理”等一系列步骤为客户构建满意的网站。

数据安全、电子防伪签章----Data security, electronic security seal

利用密码技术对图纸、通用文档、GIS数据等信息进行加密，实现要素级加密功能。包括CAD图纸加密外发软件、数据安全防泄密软件、数据共享平台加密系统、网上报建系统与电子防伪签章。

实现意义：

1. 随时追踪图纸的版权信息和来源，解决版权纠纷。

2. 防止单位内部图纸向外泄漏，保护敏感数据。

3. 检查规划设计条件（例如红线）是否被篡改、移动，保证数据的准确性。
